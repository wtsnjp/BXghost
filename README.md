# The BXghost Package

LaTeX: ghost insertion

## System requirements

* TeX format: LaTeX
* TeX engine: LuaTeX, XeTeX, pTeX, upTeX, and ApTeX (pTeX-ng)

## Usage

This package provides following commands:

```tex
\eghostguarded{<text>}
\jghostguarded{<text>}
```
